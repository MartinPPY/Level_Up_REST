# Aplicacion REST

## Como instalar la base de datos

La base de datos utilizada es la de MySQL, por lo que existen varias formas de instalarla, dejare 2 opciones viables:

### Instalar XAMPP

Descargar e instalar XAMPP y activar PHPMyadmin y Apache para la base de datos

### Docker

Instalar docker engine o Docker desktop para crear una imagen de la base de datos
